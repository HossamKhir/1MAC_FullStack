Markup

Markup is text with special meaning

it is achieved through tags "keywords within < >" that create elements

em: stands for "emphasis"

tags consist of opening (start) tag, and closing (end) tag

All whitespaces "spaces, tabs, line breaks" are collapsed into a single space

the break tag is used to break lines

Void "empty" elements: elements that has only opening tags with no closing tags
such tags are called "void tags"

the P element: for paragraphs
