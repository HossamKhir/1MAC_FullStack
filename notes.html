Markup

Markup is text with special meaning

it is achieved through tags "keywords within < >" that create elements

em tag: stands for "emphasis"

strong tag

tags consist of opening (start) tag, and closing (end) tag

All whitespaces "spaces, tabs, line breaks" are collapsed into a single space

the break tag is used to break lines

Void "empty" elements: elements that has only opening tags with no closing tags
such tags are called "void tags"

the P element: for paragraphs

Mark tag

Sup tag (superscript) Normal text Sub tag (subscript)

Nesting: is to include some tags inside other tags while maintaining the order of the tags. (i.e. elements cannot overlap, the last open tag must be closed before the one preceding it)

The paragraph element is a block element as it creates a block
whilst the em, strong, and even br are called inline elements as they don't create any blocks

Examples of block elements:
p, ol(ordered list), ul(unordered list), hN(headings; N=1~6), video, div, table

Implied Closed Tags

the img (image) tag: a void element
it requires 2 attributes:

  1. src "source:the link of the image"
  2. alt "alternate:text description of the image"
the alt attribute insturcts the browser to display its content in case of failure of getting the image

Relative URL: the URL that includes no "://" and refer to something on the local m/c
