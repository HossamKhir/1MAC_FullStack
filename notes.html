Markup

Markup is text with special meaning

it is achieved through tags "keywords within angular brackets" that create elements

em tag: stands for "emphasis"

strong tag

tags consist of opening (start) tag, and closing (end) tag

All whitespaces "spaces, tabs, line breaks" are collapsed into a single space

the break tag is used to break lines

Void "empty" elements: elements that has only opening tags with no closing tags
such tags are called "void tags"

the P element: for paragraphs

Mark tag

Sup tag (superscript) Normal text Sub tag (subscript)

Nesting: is to include some tags inside other tags while maintaining the order of the tags. (i.e. elements cannot overlap, the last open tag must be closed before the one preceding it)

Implied Closed Tags

the img (image) tag: a void element
it requires 2 attributes:

  1. src "source:the link of the image"
  2. alt "alternate:text description of the image"

the alt attribute insturcts the browser to display its content in case of failure of getting the image

Relative URL: the URL that includes no "://" and refer to something on the local m/c

the DOCTYPE tag: indicates what type of HTML the browser is dealing with

the "html" tag can have attributes, like:
"lang" for language

The browsers create a data structure to interpret the HTML source into the view in web browser tab:
the data structure is a tree structure, and it is called Document Object Model (DOM)

Tree Structure:

contains:

  • Nodes
  • Branches

rules:

  • A tree must have a root
  • Each node has branches to its children
  • Each node has only one parent, except root

Style tag:
added in the head section, and it describes the style an element should have
\ it can be applied either:

  • inline
  • internal
  • external

When working with attributes, you may use multiple values within quotes
However, the quotes could be removed, provided that the values are not separated by white spaces, otherwise, the browser would treat it as another attribute

CSS syntax:
ruleset:

  • Selector
  • Declaration block:
    • Property
    • Value
CSS selectors:
  • Type selector (p, em, etc...)
  • Class selector (.class_name), useful for several elements
  • ID selector (#id_name), useful with a single element
  • Descendant selectors (two or more type selectors separated with a space, the first is the parent, the rest are nested children)
Units:
  • Pixels
  • Points (usually with fonts)
  • em "M"; a relative unit = font-size

Boxes have:

  • Width
  • Height
  • Padding (space inside the box)
  • Border
  • Margin (space outside the box)

The link tag:
inserted into the head of the html page. It is commonly use to link to CSS files.
It usually has 2 attributes:

  • rel (relation)
  • href (reference)

Fonts can be altered using styling:
Properties like:

  • Font family
  • Font size
  • Font weight (bold)
  • Font style (italic)
  • Text decoration (underline)
